`bin/linux/codex_prompts/` contains runnable shell wrappers for Codex-driven prompt workflows.

Organization:
- `monetary/` for money-event and monetary-analysis workflows
- `personality/` for open-coding and final personality-profile workflows
- `summaries/` for stream-summary workflows

Compatibility:
- Top-level `.sh` files are lightweight forwarding shims.
- The actual maintained wrappers live in the category subfolders.
