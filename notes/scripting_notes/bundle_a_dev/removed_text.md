
# Start Here: How To Use This Report
This report is designed to support quick, practical decisions for managers and talent teams.  
Use the floating table of contents to jump directly to the section that matches the decision you need to make now.

## Quick Paths by Role
1. **Manager fast scan (5-10 minutes):** Overall Performance Snapshot -> Trends Over Time -> Collaboration Effectiveness -> Wrap-up.
2. **Talent optimization path:** Engagement Distribution -> Timing Effects -> Topic Performance -> Tag Performance.
3. **Audience strategy path:** Audience Age + Gender Trends.

## How to Interpret Responsibly
- This report is **observational**, not causal. It shows patterns, not proof of cause-and-effect.
- Tag and topic metrics can be influenced by timing, promotion, seasonality, and posting volume.
- Tag totals are directional because tags can overlap across videos.

## Standard vs Customized Content
The structure and chart explanations are standardized for consistency across talents.  
Use the **Wrap-up** section to add talent-specific narrative and recommendations.
