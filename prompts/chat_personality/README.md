# Chat Personality Prompts

This folder mirrors `bin/linux/codex_prompts/chat_personality/`.

Purpose:

- Keep chat-side personality prompts separate from `streamer_personality/`.
- Treat chat as its own community-facing analytic object, not as evidence for streamer personality unless a workflow explicitly asks for streamer uptake.
- Write derived talent-local chat outputs under each talent's `qualitative coding/chat data` folder.

Maintained prompts:

- `chat_personality_open_coding.md`
- `chat_shared_behavior_baseline.md`
- `chat_personality_unique_features.md`
- `chat_personality_qualitative_codebook.md`
- `chat_personality_profile.md`
