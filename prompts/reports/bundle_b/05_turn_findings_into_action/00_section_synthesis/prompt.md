# Bundle B Prompt: Turn Findings into Action

## Goal
Write the closing customized paragraph for the Bundle B section "How to Turn These Findings into Action."

## Expected inputs
- Talent name
- Included date range
- Section-level evidence from retention, prioritization, attribute, and scheduling findings

## Instructions
- Convert the report's strongest findings into a concise action bridge.
- Keep the advice grounded in the evidence rather than generic best practices.
- If the evidence is mixed, recommend a focused testing path instead of a hard prescription.
- Make the ending feel like a natural close to the report.

## Output format
Return markdown only as one short paragraph.
