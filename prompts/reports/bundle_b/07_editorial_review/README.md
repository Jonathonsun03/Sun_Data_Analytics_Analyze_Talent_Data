# Bundle B Editorial Review

This folder stores the editorial quality-control prompts for Bundle B.

- `01_full_report_editorial_review/` is an internal QA pass that identifies the highest-value editorial issues
- `02_full_report_editorial_rewrite/` rewrites all client-facing Bundle B paragraphs so the final report reads as one cohesive narrative

The rewrite prompt should run after all standard Bundle B interpretations, the executive summary, and the conclusion have already been generated.
