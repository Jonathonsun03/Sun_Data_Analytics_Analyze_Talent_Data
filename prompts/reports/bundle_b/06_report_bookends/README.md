# Bundle B Report Bookends

This folder stores the report-level opening and closing prompts for Bundle B.

- `01_executive_summary/` writes the top summary paragraph shown near the start of the report
- `02_conclusion/` writes the closing paragraph shown at the end of the report

These prompts should be generated after the section-level and chart-level Bundle B paragraphs exist, because they depend on the report's completed narrative.
