# Bundle A Prompt: Total Revenue by Content Type

## Section key
`total_revenue_by_content_type`

## Goal
Write a concise interpretation for the Bundle A section "Total Revenue by content type."

## Expected inputs
- Talent name
- Included date range
- Parent section: Overall Performance Snapshot
- Existing section text:
  - what this chart shows
  - how to read it
- Publish-month cohort revenue summary by content type
- Paired revenue table if available

## Instructions
- Interpret the chart as a publish-month cohort view, not cash collected inside a single month.
- Identify which content types appear to contribute most to lifetime revenue across cohorts.
- Note whether the mix is concentrated in a small number of formats or more broadly distributed.
- If a recent cohort looks incomplete or immature, mention that carefully when the input supports it.
- Avoid claiming that one format caused the revenue outcome.

## Output format
Return markdown only as one short paragraph.

Use 4-5 sentences in a concise narrative style.
Follow the same Bundle A narrative pattern used across all prompts:
- naturally include the interpretation first
- then explain why that pattern matters for monetization planning
- end with a natural recommended follow-up
Do not use bullets, headings, labels, or repetitive phrasing.
