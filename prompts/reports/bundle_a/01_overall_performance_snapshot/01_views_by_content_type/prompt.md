## Section key
`views_by_content_type`

## Goal
Write a concise interpretation for the Bundle A section "Views by Content Type (Total vs Average)."

## Expected inputs
- Talent name
- Included date range
- Parent section: Overall Performance Snapshot
- Existing section text:
  - what this chart shows
  - how to read it
- Chart summary or table for total views by content type
- Chart summary or table for average views per video by content type

## Instructions
- Compare scale and efficiency separately.
- Identify whether any content type is strong on total output but weaker on per-video efficiency, or vice versa.
- Highlight the clearest ranking or gap only if it is visibly supported by the data.
- If differences are small or mixed, say that plainly.
- Do not claim causation.
- Do not invent reasons that are not supported by the provided inputs.

## Output format
Return markdown only as one short paragraph.

Use 4-5 sentences in a concise narrative style.
Follow the same Bundle A narrative pattern used across all prompts:
- naturally include the interpretation first
- then explain why that pattern matters for planning
- end with a natural recommended follow-up
Do not use bullets, headings, labels, or repetitive phrasing.
