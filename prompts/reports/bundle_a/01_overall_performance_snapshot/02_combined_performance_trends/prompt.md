# Bundle A Prompt: Combined Performance Trends (Views + Revenue)

## Section key
`combined_performance_trends`

## Goal
Write a concise interpretation for the Bundle A section "Combined Performance Trends (Views + Revenue)."

## Expected inputs
- Talent name
- Included date range
- Parent section: Overall Performance Snapshot
- Existing section text:
  - what this chart shows
  - how to read it
- Time-series summary for views over time
- Time-series summary for revenue over time
- Any paired month-by-month table if available

## Instructions
- Describe the overall direction across the reporting window.
- Call out turning points, acceleration, flattening, or divergence between views and revenue only if clearly visible.
- Note whether the pattern looks steady, volatile, or uneven.
- If views and revenue move differently, describe the mismatch without overexplaining it.
- Keep the interpretation descriptive, not causal.

## Output format
Return markdown only as one short paragraph.

Use 4-5 sentences in a concise narrative style.
Follow the same Bundle A narrative pattern used across all prompts:
- naturally include the interpretation first
- then explain why that pattern matters operationally
- end with a natural recommended follow-up
Do not use bullets, headings, labels, or repetitive phrasing.
