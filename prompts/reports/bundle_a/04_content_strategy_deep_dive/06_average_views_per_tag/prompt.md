# Bundle A Prompt: Average Views per Tag

## Section key
`average_views_per_tag`

## Goal
Write a concise interpretation for the Bundle A section "Average Views per Tag."

## Expected inputs
- Talent name
- Included date range
- Parent section: Content Strategy Deep Dive
- Existing section text:
  - what this chart shows
  - how to read it
- Average views per tag summary
- Paired tag table if available

## Instructions
- Focus on per-upload efficiency, not total volume.
- Identify which tags appear strongest on average views per tagged video.
- If a tag is strong on average but likely driven by few videos, be careful and say so when the input supports it.
- Contrast efficiency language here with total contribution language from the broader tag section.
- Keep the interpretation practical and conservative.

## Output format
Return markdown only as one short paragraph.

Use 4-5 sentences in a concise narrative style.
Follow the same Bundle A narrative pattern used across all prompts:
- naturally include the interpretation first
- then explain why that pattern matters for future packaging choices
- end with a natural recommended follow-up
Do not use bullets, headings, labels, or repetitive phrasing.
