# Bundle A Prompt: Timing Effects - Day-of-Week Distribution

## Section key
`day_of_week_distribution`

## Goal
Write a concise interpretation for the Bundle A section "Timing Effects: Day-of-Week Distribution."

## Expected inputs
- Talent name
- Included date range
- Parent section: Content Strategy Deep Dive
- Existing section text:
  - what this chart shows
  - how to read it
- Day-of-week share summary for views and revenue
- Paired table if available

## Instructions
- Focus on contribution share by weekday.
- Identify which weekdays account for the biggest share of views or revenue.
- Note whether the pattern is concentrated in a small number of days or spread across the week.
- If views and revenue peak on different days, mention that explicitly.
- Avoid interpreting share as proof that a weekday is inherently better.

## Output format
Return markdown only as one short paragraph.

Use 4-5 sentences in a concise narrative style.
Follow the same Bundle A narrative pattern used across all prompts:
- naturally include the interpretation first
- then explain why that pattern matters for weekly programming
- end with a natural recommended follow-up
Do not use bullets, headings, labels, or repetitive phrasing.
