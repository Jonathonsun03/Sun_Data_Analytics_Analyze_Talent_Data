# Bundle A Prompt: Audience Age + Gender Trends

## Section key
`audience_age_gender_trends`

## Goal
Write a concise interpretation for the Bundle A section "Audience Age + Gender Trends."

## Expected inputs
- Talent name
- Included date range
- Parent section: Audience Composition
- Existing section text:
  - what this chart shows
  - how to read it
- Audience share trend summary by age and gender segment
- Paired table if available

## Instructions
- Focus on changes in audience composition, not on absolute audience size.
- Identify the segments that look most stable, expanding, or contracting.
- Call out concentration only when one or two segments clearly dominate.
- If the mix is broadly stable, say that directly.
- Avoid overinterpreting small movements as meaningful audience shifts.

## Output format
Return markdown only as one short paragraph.

Use 4-5 sentences in a concise narrative style.
Follow the same Bundle A narrative pattern used across all prompts:
- naturally include the interpretation first
- then explain why that pattern matters for positioning or targeting
- end with a natural recommended follow-up
Do not use bullets, headings, labels, or repetitive phrasing.
